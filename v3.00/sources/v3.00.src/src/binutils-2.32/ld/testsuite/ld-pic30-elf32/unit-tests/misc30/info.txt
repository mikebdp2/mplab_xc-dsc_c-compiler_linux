misc30: Decoding DWARF records with objdump:

The objective of this test is to verify that
DWARF debugging records in linked object files
are correctly decoded (with no errors reported.)
Both classic dsPIC and 33A formats are tested.

How to execute the test:

1. Set the environment variable $PIC30_CD
    to your pic30 tools binary directory
2. Export $PIC30_CD
3. Execute "go"

To get verbose output:

Execute "go -v"

File Descriptions:
-----------------
